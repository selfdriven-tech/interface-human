Orchestrator Distribute Voting Transaction for Witnessing

  • Building the governance action into selfdriven-gov-transaction-[govactionid].unsigned
  • cardano-cli transaction txid --tx-body-file selfdriven-gov-transaction-[govactionid].unsigned > selfdriven-gov-transaction-[govactionid].hash
  • Put the selfdriven-gov-transaction-[govactionid].hash file into the /members/voting folder
  • Let the members know they can get the file and witness it using the selfdriven-gov App 1.0.4 or later.
